🚀 **الخطوة الوحيدة المطلوبة:**

انقر نقراً مزدوجاً على الملف:
   → رفع_الكود.bat

هذا كل شيء! الملف سيقوم تلقائياً بـ:
   ✅ إضافة جميع الملفات
   ✅ إنشاء commit
   ✅ رفع الكود إلى GitHub

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📋 **بعد رفع الكود:**

1. سيتم فتح GitHub Actions تلقائياً
   (أو افتح يدوياً: https://github.com/qutibalwaabi/aldafary-app/actions)

2. اضغط على "Build iOS" من القائمة

3. اضغط "Run workflow" → "Run workflow"

4. انتظر 5-10 دقائق حتى يكتمل البناء

5. اضغط على "ios-app" في قسم Artifacts

6. حمّل ملف .ipa

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

💡 **إذا واجهت مشكلة:**

استخدم Source Control في Cursor:
   1. اضغط Ctrl+Shift+G
   2. اضغط على "..." في الأعلى
   3. اختر "Push"

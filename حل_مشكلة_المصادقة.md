# حل مشكلة مصادقة GitHub

## المشكلة:
عند الضغط على "Sign in with your browser" لا يحدث شيء.

## الحلول:

### الحل 1: استخدام Token (الأسهل والأسرع)

1. اذهب إلى: https://github.com/settings/tokens
2. اضغط "Generate new token (classic)"
3. اختر الصلاحيات: **repo** (كلها)
4. انسخ الـ Token
5. عندما يطلب منك كلمة المرور، الصق الـ Token

### الحل 2: استخدام "Sign in with a code"

1. اضغط على "Sign in with a code"
2. سيعطيك رمز
3. افتح المتصفح واذهب إلى: https://github.com/login/device
4. أدخل الرمز
